# Motercontroller_vol1

## 何するもの？
* VNH5019搭載のモーターコントローラの内部のプログラム

### CAN通信について
* IDはすべて16進数でなく10進数で表現されています

#### コントローラーID
0～7の8段階で設定可能
変更時のデータ(マスター -> スレーブ)

ID : 199

data[0]:新しいID(0～7)
data[1～7]:0

#### モーターENABLE
これを送信しないとENABLEにならない

ID : 110 + コントローラーID

data[0～7]:0

#### モーターDISABLE

ID : 120 + コントローラーID

data[0~7]:0
